# netBOY GitHub Pages Release

This project contains the static release build for serving netBOY from GitHub Pages.

## Deploy

1. Create a new GitHub repository for this folder.
2. Commit and push all files in this directory.
3. In GitHub, open Settings > Pages.
4. Set the source to the default branch and the root directory.

The release was built with the COI service worker mode so mGBA can run on static hosting that cannot set COOP/COEP headers.
